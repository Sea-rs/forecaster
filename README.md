# FORECASTER (Step 1)

PHP + JSONで構成した最小実装です。

## 実装済み（Step 1）

- TOPページ
- CSV取り込みページ
- Viewページ（取り込みデータ表示）
- JSONファイル保存

## ディレクトリ

- public/: 公開ページ
- src/: アプリ内部ロジック
- data/records.json: 擬似DB

## 起動方法（ローカル確認）

1. プロジェクト直下で実行

   php -S localhost:8000 router.php

2. ブラウザで開く

   http://localhost:8000

## 現時点の仕様

- CSVの1行目をヘッダーとして扱います。
- 2行目以降をJSONに追記保存します。
- 空行は取り込みません。
- 同じデータの重複判定はまだ行いません。

## 次の拡張候補（Step 2以降）

- 取り込み時バリデーション（必須列、日付形式、数値形式）
- 重複チェック
- Viewページの検索・並び替え・ページング
- 取り込み履歴管理
- エラーログ表示
